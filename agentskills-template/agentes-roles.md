# Agentes/Roles no Projeto

- Product Analyst: elicita requisitos, refina épicos, faz perguntas críticas
- Tech Lead: avalia impacto técnico, define arquitetura, revisa decisões de alto risco
- QA/Tester: define critérios de aceitação, casos de borda, valida fluxos
- UX/UI: desenha jornadas, wireframes, identifica gargalos de usabilidade
- Dev Backend: implementa regras, integrações, garante consistência
- Dev Frontend: implementa telas, componentes, garante UX
- PO/Cliente: prioriza, responde perguntas de negócio, aprova entregas

**Como usar:**
- Para cada decisão, avalie qual agente/role deve ser envolvido
- Use este guia para alinhar responsabilidades e expectativas
