# Agents & Skills por Área do Projeto

Mapeie rapidamente quais skills e agentes/roles são críticos para cada área do seu produto. Use como checklist para planejamento, review ou onboarding de novos membros.

---

## Frontpage / Landing Page / Dashboard
- **Skills:**
  - Design de UX/UI
  - Composição de componentes
  - Navegação e fluxo
  - Acessibilidade
  - Escrita de mensagens de valor
- **Agents:**
  - UX/UI
  - Dev Frontend
  - PO

## Backend / API / Actions / Infraestrutura
- **Skills:**
  - Modelagem de dados
  - Implementação de regras de negócio
  - Segurança e autenticação
  - Testes automatizados
  - Integração com sistemas externos
- **Agents:**
  - Tech Lead
  - Dev Backend

## Agendamento
- **Skills:**
  - Algoritmo de disponibilidade
  - Detecção de conflitos de agenda
  - Suporte a múltiplos profissionais
  - Idempotência (robô/integração)
- **Agents:**
  - Product Analyst
  - Dev Backend
  - QA

## Financeiro / Caixa
- **Skills:**
  - Cálculo de comissão
  - Fechamento diário
  - Estorno/reembolso
  - Relatórios financeiros
- **Agents:**
  - Product Analyst
  - Dev Backend
  - QA

## Estoque / Produtos
- **Skills:**
  - Controle de inventário
  - Gestão de variações/SKUs
  - Ajustes e logs de estoque
- **Agents:**
  - Dev Backend
  - QA

## Relatórios / Analytics
- **Skills:**
  - Definição de métricas
  - Geração de relatórios customizados
  - Visualização de dados
- **Agents:**
  - Product Analyst
  - Dev Backend
  - Dev Frontend
