# AgentSkills Template

Este diretório contém um template reutilizável para criação, revisão e expansão de projetos digitais, especialmente focado em produtos complexos (ex: sistemas de agendamento, financeiro, ERP, etc).

Inclui:
- Estrutura de PRD/épico reutilizável
- Lista de skills/heurísticas para elicitação e análise
- Guia de perguntas poderosas
- Manual de agentes/roles
- Instruções de uso

**Como usar:**
- Copie esta pasta para o novo projeto
- Use o PRD/épico como espinha dorsal
- Adapte as perguntas e skills conforme o contexto
- Compartilhe com o time para alinhar raciocínio e decisões
