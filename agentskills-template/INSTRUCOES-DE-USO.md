# Instruções de Uso do AgentSkills Template

1. Copie a pasta `agentskills-template` para o novo projeto.
2. Use o arquivo `PRD-Template.md` como base para especificação de novos produtos ou revisão de antigos.
3. Consulte `skills-heuristicas.md` e `perguntas-poderosas.md` para destravar requisitos e decisões.
4. Use `agentes-roles.md` para alinhar papéis e responsabilidades no time.
5. Compartilhe este template com outros projetos para padronizar a abordagem de análise, modelagem e documentação.
6. Adapte e expanda conforme o contexto do projeto.
